# Agent Workbench Language

agent 工作台的产品语言记录。这里保存用户在工作台里直接面对的核心概念，帮助协作者区分对话、引用、图谱、抽屉和产出之间的边界。

## Language

**当前知识库**:
用户此刻在工作台中选中的知识库。对话、引用、查询、图谱和对话结晶都围绕当前知识库发生。
_Avoid_: 全部知识库, 默认知识库, 当前目录

**外部知识库**:
用户手动登记到工作台里的知识库，位置不在默认知识库根目录下。它仍然必须符合 llm-wiki 的知识库格式。
_Avoid_: 外部数据源, 远程知识库

**对话**:
用户和 agent 围绕一个知识库展开的一条工作线程。对话可以引用页面、执行命令、生成产出物，并在需要时通过对话结晶保存为 wiki 页面。
_Avoid_: 聊天记录, 页面, 会话文件

**当前对话**:
当前知识库里正在打开的那条对话。切换知识库时，当前对话也随之切换到目标知识库自己的对话列表。
_Avoid_: 最近对话, 全局对话

**引用**:
把当前知识库里的 wiki 页面带入对话的动作。引用回答“基于哪些内容说”，不表示执行工具。
_Avoid_: 命令, 搜索, 附件

**命令**:
在对话里触发一项能力的动作，例如消化、对话结晶、导出或调用已加载的 Skill。命令回答“要做什么”，不表示引用某个页面。
_Avoid_: 引用, 页面链接

**对话结晶**:
把当前工作台对话中有保存价值的内容整理成 `wiki/synthesis/sessions/` 下的 wiki 页面。它可以读取当前对话和工作台状态；不要和 Skill 的手动结晶 / `crystallize` 混用。
_Avoid_: 手动结晶, 聊天记录导出, 产出预览

**右抽屉**:
工作台右侧的辅助阅读和预览区域。它承载 wiki 页面阅读、图谱摘要、选区提问、产出物预览和设置，不替代主对话。
_Avoid_: 主屏, 新页面, 编辑器

**产出预览**:
在右抽屉中查看产出物的状态。预览帮助用户确认和下载结果，但产出物本身不自动成为 wiki 页面。
_Avoid_: wiki 页面阅读, 对话结晶

**活地图**:
工作台中的知识图谱视图。它不是静态图片，而是可以响应当前知识库变化、选择页面组并把页面组带回对话的地图。
_Avoid_: 导出图, 装饰图

**活地图阅读流程**:
用户在活地图中从全局浏览进入局部阅读、打开右抽屉、形成选区并把页面组带回对话的连续流程。它描述的是用户阅读和提问的路径，不是单个组件、动画或图谱算法。
_Avoid_: 图谱组件, 图谱动画, App 内部状态

**社区**:
图谱中围绕相近关系聚在一起的一组 wiki 页面。社区帮助用户从局部阅读知识库，不等同于文件夹、标签或永久分类。
_Avoid_: 文件夹, 标签, 主题页

**选区**:
用户在图谱里临时选中的一组页面和关系。选区可以带入对话继续提问，但不会直接改变知识库内容。
_Avoid_: 社区, 搜索结果, 永久收藏

**社区阅读**:
从全局图谱进入某个社区后的局部阅读状态。它只聚焦当前社区内部页面和关系，帮助用户近距离理解这一组内容。
_Avoid_: 全局图谱, 主题页

**批量消化**:
在工作台里一次处理多个可消化文件的流程。它用于把一批原始素材整理进当前知识库，而不是创建一个新的知识库。
_Avoid_: 新建知识库, 全库重建
