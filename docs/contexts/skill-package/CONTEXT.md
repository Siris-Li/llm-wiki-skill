# Skill Package Language

Skill 形态的项目语言记录。这里保存 `SKILL.md`、`scripts/`、`templates/` 和各平台入口共用的工作流概念，帮助协作者区分建库、入库、查询、综合、检查和删除等动作。

## Language

**工作流路由**:
根据用户意图把请求分配到 init、ingest、query 等具体工作流的判断过程。路由只决定下一步走哪条流程，不直接修改知识库。
_Avoid_: 工作流执行, 命令菜单

**通用前置检查**:
除 init 外多数工作流开始前确认知识库位置、读取配置和判断输出语言的步骤。它保证后续动作落在正确知识库上。
_Avoid_: 初始化, 状态检查

**init**:
创建一个新的 llm-wiki 知识库，并生成目录结构、配置、索引、日志、研究方向和缓存文件。
_Avoid_: ingest, 添加素材, 新建页面

**ingest**:
把单个原始素材保存进知识库，并整理出素材摘要页、实体页、主题页和相关索引记录的工作流。
_Avoid_: query, digest, 文件导入

**batch-ingest**:
对一个文件夹或一批文件逐个执行 ingest 的工作流。它负责批量调度和进度汇总，不改变单个素材的处理规则。
_Avoid_: ingest, 新建知识库

**query**:
基于已有 wiki 页面快速回答用户问题的工作流。query 默认是即时回答，只有综合价值足够时才建议保存为 query 页面。
_Avoid_: digest, ingest, 全网搜索

**digest**:
跨多个已有 wiki 页面生成持久化综合报告的工作流。digest 会写入综合分析、对比或时间线页面，不只是回答一句问题。
_Avoid_: query, ingest, 临时总结

**lint**:
检查知识库健康状况的工作流，包含机械检查和 AI 判断类检查。lint 负责发现孤立页面、断链、矛盾、缺失引用和置信度问题。
_Avoid_: status, graph, 自动重写

**status**:
汇总知识库当前规模、素材分布、页面数量、最近活动和外挂状态的工作流。status 只报告现状，不做修复。
_Avoid_: lint, query, graph

**graph**:
从 wiki 页面链接和关系信号生成知识图谱文件的工作流。graph 负责生成 Mermaid 图、交互式图谱数据和离线 HTML，不负责解释所有内容。
_Avoid_: digest, 图谱阅读, query

**delete**:
删除某份原始素材及其素材摘要页，并清理相关引用、索引、日志和缓存的工作流。delete 是级联清理，不只是删除一个文件。
_Avoid_: 隐藏素材, 取消引用, lint 修复

**手动结晶 / crystallize**:
把用户主动提供的一段有价值内容整理成 `wiki/synthesis/sessions/` 下页面的工作流。它可以记录对话或思考成果，但不会自动读取工作台当前会话；需要当前会话状态时，属于工作台的对话结晶。
_Avoid_: ingest, query 保存, 对话结晶, 自动读取当前会话

**来源总表**:
定义素材来源、匹配规则、raw 目录、提取器和回退提示的表。所有素材路由都应从来源总表读取，不应临时手写域名判断。
_Avoid_: 域名白名单, 提取器配置

**可选提取器**:
用于自动提取网页、X/Twitter、微信公众号、YouTube 或知乎内容的附加能力。可选提取器失败时，核心知识库流程仍应能通过手动入口继续。
_Avoid_: 核心依赖, 必装工具

**手动入口**:
自动提取不可用、不支持或结果为空时，让用户直接粘贴文本或提供本地文件继续处理的入口。
_Avoid_: 失败状态, 放弃处理

**外挂状态**:
可选提取器在当前环境下的可用性分类，例如未安装、环境不可用、运行失败、不支持或空结果。外挂状态用于生成可执行的回退提示。
_Avoid_: 错误码, 异常栈

**缓存命中**:
原始素材内容没有变化，且已有对应素材摘要页可复用的状态。缓存命中时不应再次调用模型重新整理同一份素材。
_Avoid_: 跳过处理, 删除缓存

**完整处理**:
长素材进入 ingest 后的标准整理路径，会生成素材摘要页，并创建或更新实体页、主题页、索引和日志。
_Avoid_: 简化处理, digest

**简化处理**:
短素材进入 ingest 后的轻量整理路径，只生成摘要页、提取少量关键概念并更新基础索引和日志。
_Avoid_: 完整处理, 跳过处理

**置信度标注**:
在素材分析和页面关系中标明信息来源可靠性的做法，例如 EXTRACTED、INFERRED、AMBIGUOUS 和 UNVERIFIED。置信度标注帮助后续检查和图谱表达。
_Avoid_: 关系类型, 重要性
